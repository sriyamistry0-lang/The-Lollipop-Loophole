# The-Lollipop-Loophole
Analyzing the Survival Kit of S. pneumoniae: A Pathogen-Centric Genomic Pipeline.
Cloud-based annotation workflow using Micromamba and Prokka to bypass dependency conflicts and parse raw bacterial assemblies.
Successfully benchmarked the pipeline on Streptococcus pneumoniae Reference Sequences, automating the extraction of key AMR drivers (Penicillin-Binding Proteins) and virulence factors (Pneumolysin, Autolysin)
Informing large-scale population genomic studies.
